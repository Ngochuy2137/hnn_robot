# Hnn Robot Description

Robot description files (URDF/xacro) for Hnn robots.

## Package Structure

This package contains:
- **URDF/Xacro files**: Robot model definitions
- **ros2_control configuration**: Hardware interface definitions in xacro format

## Usage

This package is typically used through launch files in `hnn_robot_bringup` package.
